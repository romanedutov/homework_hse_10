# homework_hse_10
CVE-2025-2294 — это зарегистрированная уязвимость безопасности в плагине Kubio Page Builder для WordPress, классифицированная как Local File Inclusion (LFI), которая возникает из-за недостаточной валидации входных данных в параметрах __kubio-site-edit-iframe-preview и __kubio-site-edit-iframe-classic-template, позволяя неавторизованному злоумышленнику осуществлять чтение произвольных файлов на файловой системе сервера через path traversal-последовательности.
